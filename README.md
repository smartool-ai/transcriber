# Transcriber

## Description

This is a Python project that uses several libraries such as `boto3`, `botocore`, `openai`, and `pynamodb` to perform certain tasks. The project is designed to generate tickets from a transcript. It downloads a transcript from an S3 bucket, generates tickets using OpenAI's GPT-4 model, and then saves these tickets to a DynamoDB table.

## Installation

To install the dependencies for this project, you will need Python and pip installed on your machine. Then, you can clone the repository and install the dependencies using Poetry.


```bash
git clone <repository>
cd <repository>
pip install boto3 botocore openai pynamodb
```
